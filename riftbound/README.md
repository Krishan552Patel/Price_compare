# Riftbound

This directory is a placeholder for the **Riftbound** project — a companion module to the FAB Cards price comparison tool, intended to support the Riftbound trading card game.

## Overview

Riftbound is a trading card game. This module will provide:

- Card data ingestion scripts for Riftbound cards
- Price scraping and comparison for Riftbound card singles
- Database schema and migration scripts for Riftbound card data
